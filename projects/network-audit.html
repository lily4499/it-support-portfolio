Home Network Audit

Performed a structured security audit of a home network — scanning for devices, identifying risks, and implementing VLAN segmentation for IoT devices.

Tools Used Nmap, Wireshark, Router Admin Panel
Duration 1 week
Scope Personal home network (authorized)

Why This Project

Network troubleshooting is a core help desk skill. Before diagnosing user network problems, you need to understand what a healthy network looks like — and what a vulnerable one looks like. I audited my own home network to build this foundational awareness.

Phase 1: Device Discovery

Used Nmap to scan the local subnet and identify every connected device:

Phase 2: Risk Identification

Cross-referenced each device and flagged three risk categories:

  • Router still using default admin credentials (critical)
  • 3 IoT devices (TV, thermostat, bulbs) on same network as PCs
  • 1 unknown device identified as a neighbor's phone (misconfigured guest network)
  • UPnP enabled on router — unnecessary exposure

Phase 3: Remediations Applied

  • Changed router admin password to a 20-character random passphrase
  • Created a separate IoT VLAN (192.168.10.0/24) and moved all smart devices to it
  • Disabled UPnP and remote management on router
  • Fixed guest network isolation settings

Wireshark Capture

Ran a 10-minute Wireshark capture to identify anomalous traffic. Observed the smart TV making periodic connections to Samsung telemetry servers — expected behavior, but documented as a baseline.

✓ Key Outcomes

  • Documented a full network inventory with MAC addresses and device roles
  • Reduced attack surface by isolating IoT devices via VLAN
  • Practiced Nmap and Wireshark in a safe, authorized environment
  • Produced a written audit report in a professional IT format
