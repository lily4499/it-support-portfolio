## 🚀 How to Host on GitHub Pages

1. Create a new GitHub repository named `it-support-portfolio`
2. Upload all files (maintaining the folder structure above)
3. Go to **Settings → Pages**
4. Under **Source**, select `main` branch and `/ (root)` folder
5. Click **Save** — your site will be live in ~60 seconds

## ✏️ How to Customize

- Replace `Alex Rivera` with your name throughout `index.html`
- Update contact links (email, LinkedIn, GitHub) in the contact section
- Edit the stats in the hero section to reflect your real numbers
- Add or remove skill cards in the skills section
- Update certifications with your actual credentials
- Swap project write-ups with your own lab work

## 📦 No Dependencies Required
Pure HTML + CSS + JavaScript. No build tools, no npm. Just open `index.html` in a browser or push to GitHub Pages.
